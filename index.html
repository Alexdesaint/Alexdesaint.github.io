Projects overview

Wiring harness viewer for augmented reality glasses

Apprenticeship at Safran

At Safran Electrical & Power design office, I had to assess the Hololens 2 glasses to improve the industrial use in the plane cabling process. The purpose of this R&D project was to use the augmented reality glasses in the wiring harness manufacture. Instead of wiring on printed schematic plans, technicians could directly visualise the schematic in the glasses.

In order to read the schematic sent by the design offices, I had to create a software able to translate 2D vectorial drawings into 3D space for augmented reality. This led to new features like path finding in the wire network or decomposing the plan into several layers.

Open source graphics engine

Alexdesaint/BlobEngine

In 2018 I started this project for learning purposes. I put in practice the 3D kinematics and C++ programming learned in robotics classes. With C++, CMAKE and OpenGL 4.5 I am coding this engine with simplicity in mind and with minimal dependency :

  • Dav1dde/glad to load OpenGL
  • GLFW to create the Window and handle inputs
  • ocornut/imgui to create graphical interfaces
  • nlohmann/json to load GLTF json files
  • nothings/stb to load image for texturing

In the future my purpose is to use this engine in a professional application and to continue the development with the Vulkan API.
